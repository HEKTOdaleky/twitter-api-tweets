# twitter-api-tweets

A simple koa-server with twitter module

## Getting Started

  - download [twitter-api-tweets](https://github.com/Wasya1212/twitter-api-tweets)
  - install dependencies by command prompt
  ```
  $ npm install
  ```
  - run
  ```
  $ npm start
  ```
  - Then go to the link http://localhost:3000 in your browser, and see the result.

### Development
  - To start work with frontend files run the following command:
  ```
  $ webpack
  ```
  - To start work with backend files run the following command:
  ```
  $ nodemon app
  ```
  - If you want run Webpack in production mode
  ```
  $ npm build
  ```
  - You can also run the **Webpack** and **Node** together, all you need to do is open 2 command prompts and enter the above commands.
